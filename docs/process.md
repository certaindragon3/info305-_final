### Hero Section照片选择
我为你预留了5个照片位置，你需要将照片放在以下路径：

  public/images/gallery/
  ├── dining-hall.jpg      # 传统用餐大厅
  ├── chef-work.jpg        # 厨师工作场景
  ├── chef-portrait.jpg    # 厨师肖像
  ├── interior.jpg         # 餐厅内饰
  └── cooking-process.jpg  # 烹饪过程

  照片建议尺寸：
  - 宽高比：4:5 (portrait orientation)
  - 推荐尺寸：800x1000px 或更高分辨率
  - 格式：JPG 或 PNG
